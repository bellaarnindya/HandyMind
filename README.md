# HandyMind
<img src="/app/src/main/res/mipmap-xxxhdpi/ic_launcher_round.png" align="left" width="200" hspace="10" vspace="10">
HandyMind membantu anda dalam mengorganisasikan apa yang anda pikirkan, mengingat dan membuat ide - ide baru anda. Simpan secara langsung apa yang anda pikirkan menjadi mind map, ubah secara langsung untuk membantu anda mempresentasikan ide anda, bagikan ide - ide anda kepada orang lain dengan mudah.</br>
HandyMind sekarang tersedia di Play Store, anda dapat memeriksanya [di sini](https://play.google.com/store/apps/details?id=air.com.EXswap.Mindomo)!</br>
